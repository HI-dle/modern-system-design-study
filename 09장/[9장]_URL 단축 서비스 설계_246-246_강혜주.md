# 9.2 API 설계

단축 URL의 핵심 기능을 구현하려면 다음 API가 필요하다.

- **POST `/shorturl`**
    - 긴 URL을 입력받아 새로운 단축 URL을 생성한다.
    - 요청 본문: 긴 URL
    - 응답: 단축 URL
    - 함수 예시: `getShortUrl(user_id, longurl)`

- **GET `/shorturls/{shorturl}`**
    - 단축 URL에 해당하는 원본 URL을 반환한다.
    - 응답: 긴 URL
    - 함수 예시: `getLongUrl(shortUrl)`

이제 대략적으로 제시해서 문제 규모를 좀 더 구체적으로 파악해 보자.
