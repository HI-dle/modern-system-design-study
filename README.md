## 📚 Github Convention

#### 발표 자료
- 파일명(파일 위치 포함)은 아래와 같이 해주세요.
```text
0X장/[0X장]_{제목}{(부가옵션)_0X-0X_소제목자유}_{이름}.md
```

#### issue
- 아래와 같이 제목을 작성해주세요.
```text
[0X장]_{제목}
```
- Issue Comment 는 제일 첫 줄에 관련 페이지 위치를 작성해주세요.
```text
p.123 n문단
~문장이 이해가 안 됩니다.
```

#### commit message
- 아래와 같이 작성해주세요.
```text
[#이슈번호]{파일이름(.md 제외)}
```
